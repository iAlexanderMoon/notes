# Git Cli Cheatsheet

### Quick look
```sh
git init
git clone {repository}
git log {filename}
git checkout {hash} -- {filename}
git checkout -b {branch}
git checkout {branch}
git pull
git push
git tag
git tag {tagname}
```

#### See commit history of a file
```sh
git log {filename}
```

#### Restore a historical file
```sh
git checkout {hash} -- {filename}
```
